# إضافة Figma — واجهات الجدول الذكي

هذه الإضافة المحلية تنشئ داخل ملف Figma واجهتين أصليتين بمقاس 430 × 932 بكسل:

- واجهة الجدول الذكي.
- واجهة المتابع الإداري.

جميع البطاقات والنصوص والخلفيات والأيقونات طبقات قابلة للتعديل وليست صورة مسطحة.

## التشغيل

1. فك ضغط الملف.
2. افتح تطبيق Figma لسطح المكتب وأنشئ ملف Design فارغًا.
3. من قائمة Plugins اختر Development ثم Import plugin from manifest.
4. اختر ملف `manifest.json` الموجود داخل المجلد.
5. شغّل الإضافة باسم **Smart Table UI Builder**.

ستُنشأ الواجهتان تلقائيًا بجانب بعضهما داخل Section واحد.

## ملاحظات

- تحاول الإضافة استخدام خط Cairo، ثم Noto Sans Arabic كبديل.
- الإضافة لا تحتاج إلى اتصال بالإنترنت.
- شغّلها مرة واحدة في الملف لتجنب تكرار الواجهات.
